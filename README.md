# Yâdigâr AI — Dijital Refakatçi (v2)

> Alzheimer hastaları ve yalnız yaşayan yaşlı bireyler için kişiselleştirilmiş yapay zekâ refakatçisi.
> Proje temeli: "Söz Uçar Robot Kalır" — HAVACILIK, UZAY VE TEKNOLOJİ FESTİVALİ, #81370

---

## 🌐 Proje Hakkında

Yâdigâr AI, hasta bilgilerini (hastalık durumu, aile bireyleri, kan grubu, kullanılmaması gereken kelimeler) öğrenerek kişiye özel sohbet eden, acil durumlarda yakınlara otomatik SMS/mesaj gönderen ve hastalık ilerleme riskini takip eden bir yapay zekâ refakatçisidir.

---

## 🚀 Nasıl Çalıştırılır

1. Cursor uygulamasını açın
2. Proje klasörünü Cursor'a sürükleyin
3. `features/index.html` dosyasını tarayıcıda açın
4. İlk kullanımda hasta bilgileri formunu doldurun
5. Google AI Studio'dan aldığınız Gemini API anahtarını ayarlara girin

---

## 🔑 Gemini API Kurulumu (Google AI Studio)

1. [aistudio.google.com](https://aistudio.google.com) adresine gidin
2. Google hesabınızla giriş yapın
3. "Get API Key" → "Create API Key" butonuna tıklayın
4. Kopyaladığınız anahtarı uygulamada **Ayarlar → API Anahtarı** alanına yapıştırın
